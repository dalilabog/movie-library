# Movie Library (Личная кинотека)

**Автор:** [Ваше Имя Фамилия]

## Описание
GUI-приложение для управления списком фильмов. Программа позволяет добавлять информацию о фильмах, фильтровать записи по жанрам и автоматически сохранять данные в формате JSON.

## Технологии
- **Python версии:** 3.8+
- **Библиотеки:** 
  - `tkinter` (интерфейс)
  - `json` (хранение данных)

## Функционал
1. **Добавление:** Поля для названия, жанра, года и рейтинга.
2. **Валидация:** Проверка на пустые поля, корректность года и диапазон рейтинга (0-10).
3. **Фильтрация:** Выбор жанра для отображения конкретных категорий.
4. **Хранение:** Автоматическое создание и чтение файла `movies.json`.

## Примеры работы (Тесты)
- **Успешный ввод:** "Inception", "Фантастика", 2010, 8.8 -> запись появляется в таблице.
- **Ошибка года:** Ввод "abc" в поле года выдаст предупреждение "Год должен быть числом".
- **Ошибка рейтинга:** Ввод "11" выдаст ошибку "Рейтинг должен быть от 0 до 10".
